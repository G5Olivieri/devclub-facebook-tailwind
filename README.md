# TAILWIND CSS - APRENDA DO ZERO - Recriando o Facebook com Tailwind CSS

https://www.youtube.com/watch?v=zVO-3yAMRs4&t=11s

# Setup

```bash
$ npm init -y
```

```bash
$ npm i -D tailwindcss
```

```bash
$ npx tailwindcss init
```

```bash
$ npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch
```
